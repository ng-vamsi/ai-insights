# Storage Implementation Guide

## Overview

This extension uses **chrome.storage.local** as the primary persistence layer, with a capacity of 10MB (10,485,760 bytes). This guide details the storage architecture, data structures, and usage patterns.

---

## Storage Capacity Analysis

### Quota
- **Total**: 10MB (QUOTA_BYTES: 10485760)
- **Check usage**: `chrome.storage.local.getBytesInUse()`
- **Per-item limit**: 8KB recommended for performance

### Estimated Capacities

| Data Type | Size per Item | Max Items | Notes |
|-----------|---------------|-----------|-------|
| Recording Session | 2-3 KB | ~3,300 - 5,000 | Depends on transcript length |
| Call History | 2-4 KB | ~2,500 - 5,000 | With AI summary |
| API Keys | <1 KB | N/A | Negligible |
| Full 10-min call | ~5-8 KB | ~1,300 - 2,000 | With full transcript + insights |

**Practical Capacity**: Approximately **5,000 short calls** or **1,500 long calls** before reaching the limit.

## Storage Operations

### Read Operations

#### Get All Sessions
```javascript
chrome.storage.local.get(null, (items) => {
  const sessions = Object.entries(items)
    .filter(([key]) => key.startsWith('session_'))
    .map(([_, value]) => value);
  console.log('Sessions:', sessions);
});
```

#### Get Latest Session
```javascript
chrome.storage.local.get(null, (items) => {
  const sessions = Object.entries(items)
    .filter(([key]) => key.startsWith('session_'))
    .map(([_, value]) => value)
    .sort((a, b) => b.lastUpdate - a.lastUpdate);
  
  const latest = sessions[0];
});
```

#### Get Call History

```javascript
chrome.storage.local.get(null, (items) => {
```

#### Get Today's Calls

### Write Operations

#### Save Session (Every 5 seconds during recording)
```javascript
function saveCurrentSession() {
  const sessionData = {
    sessionId: currentSessionId || generateSessionId(),
    startTime: recordingStartTime,
    lastUpdate: Date.now(),
    isRecording: isRecording,
    recordingDuration: Math.floor((Date.now() - recordingStartTime) / 1000),
    transcript: fullTranscript,
    questions: detectedQuestions,
    liveInsights: currentInsights
  };
  
  chrome.storage.local.set({
    [`session_${sessionData.sessionId}`]: sessionData
  }, () => {
    console.log('Session saved:', sessionData.sessionId);
  });
}

// Auto-save every 5 seconds
setInterval(saveCurrentSession, 5000);
```

#### Convert Session to Call
```javascript
function saveCallToHistory(sessionData, aiSummary) {
```

```javascript
function updateCallMetadata(callId, tags, notes) {
```

```javascript
  chrome.storage.local.get(null, (items) => {
```

#### Delete Old Calls (Keep last N)
```javascript
function deleteOldCalls(keepCount = 100) {
  chrome.storage.local.get(null, (items) => {
    const calls = Object.entries(items)
      .filter(([key]) => key.startsWith('call_'))
      .map(([key, value]) => ({ key, ...value }))
      .sort((a, b) => b.timestamp - a.timestamp);
    
    const toDelete = calls.slice(keepCount).map(c => c.key);
    
    if (toDelete.length > 0) {
      chrome.storage.local.remove(toDelete, () => {
        console.log(`Deleted ${toDelete.length} old calls`);
      });
    }
  });
}
```

## Storage Monitoring

### Check Storage Usage
```javascript
chrome.storage.local.getBytesInUse(null, (bytes) => {
  const usedMB = (bytes / 1024 / 1024).toFixed(2);
  const totalMB = 10;
  const percentUsed = ((bytes / 10485760) * 100).toFixed(1);
  
  console.log(`Storage: ${usedMB} MB / ${totalMB} MB (${percentUsed}%)`);
  
  if (percentUsed > 80) {
    console.warn('⚠️ Storage almost full!');
  }
});
```

### Storage Warning System
```javascript
function checkStorageAndWarn() {
  chrome.storage.local.getBytesInUse(null, (bytes) => {
    const percentUsed = (bytes / 10485760) * 100;
    
    if (percentUsed > 90) {

    } else if (percentUsed > 80) {

```

### Export Data (Backup)
```javascript
function exportAllData() {
  chrome.storage.local.get(null, (items) => {
    const dataStr = JSON.stringify(items, null, 2);
    const blob = new Blob([dataStr], { type: 'application/json' });
    const url = URL.createObjectURL(blob);
    
    const a = document.createElement('a');
    a.href = url;
    a.download = `sales-insights-backup-${Date.now()}.json`;
    a.click();
    
    URL.revokeObjectURL(url);
  });
}
```

### Import Data (Restore)
```javascript
function importData(file) {
  const reader = new FileReader();
  reader.onload = (e) => {
    const data = JSON.parse(e.target.result);
    chrome.storage.local.set(data, () => {
      console.log('Data imported successfully');
    });
  };
  reader.readAsText(file);
}
```

## Performance Optimization

### Batch Operations
```javascript
// ❌ BAD: Multiple writes
calls.forEach(call => {
  chrome.storage.local.set({ [`call_${call.id}`]: call });
});

// ✅ GOOD: Single batch write
const batch = {};
calls.forEach(call => {
  batch[`call_${call.id}`] = call;
});
chrome.storage.local.set(batch);
```

### Lazy Loading
```javascript
// ❌ BAD: Load all data upfront
chrome.storage.local.get(null, (items) => {
  // Process 5000 calls...
});

// ✅ GOOD: Load on-demand
function loadCallById(callId, callback) {
  chrome.storage.local.get(`call_${callId}`, (result) => {
    callback(result[`call_${callId}`]);
  });
}
```

### Pagination
```javascript
// Load calls in pages of 20
function loadCallsPage(page = 0, pageSize = 20) {
  chrome.storage.local.get(null, (items) => {
    const calls = Object.entries(items)
      .filter(([key]) => key.startsWith('call_'))
      .map(([_, value]) => value)
      .sort((a, b) => b.timestamp - a.timestamp);
    
    const start = page * pageSize;
    const end = start + pageSize;
    const pageCalls = calls.slice(start, end);
    
    return {
      calls: pageCalls,
      totalPages: Math.ceil(calls.length / pageSize),
      currentPage: page
    };
  });
}
```

## Cleanup Strategies

### 1. Time-Based Cleanup (Keep last 30 days)
```javascript
function cleanupOldCalls(daysToKeep = 30) {
  const cutoffDate = Date.now() - (daysToKeep * 24 * 60 * 60 * 1000);
  
  chrome.storage.local.get(null, (items) => {
    const oldCalls = Object.entries(items)
      .filter(([key]) => key.startsWith('call_'))
      .filter(([_, call]) => call.timestamp < cutoffDate)
      .map(([key]) => key);
    
    if (oldCalls.length > 0) {
      chrome.storage.local.remove(oldCalls, () => {
        console.log(`Deleted ${oldCalls.length} old calls`);
      });
    }
  });
}
```

### 2. Size-Based Cleanup (Keep storage under 80%)
```javascript
function cleanupBySize() {
  chrome.storage.local.getBytesInUse(null, (bytes) => {
    const percentUsed = (bytes / 10485760) * 100;
    
    if (percentUsed > 80) {
      chrome.storage.local.get(null, (items) => {
        const calls = Object.entries(items)
          .filter(([key]) => key.startsWith('call_'))
          .map(([key, value]) => ({ key, ...value }))
          .sort((a, b) => a.timestamp - b.timestamp); // Oldest first
        
        // Delete oldest 20% of calls
        const toDeleteCount = Math.ceil(calls.length * 0.2);
        const toDelete = calls.slice(0, toDeleteCount).map(c => c.key);
        
        chrome.storage.local.remove(toDelete);
      });
    }
  });
}
```

## Migration & Versioning

### Schema Version
```javascript
const STORAGE_VERSION = 2;

// Check and migrate on startup
chrome.storage.local.get('storageVersion', (result) => {
  const currentVersion = result.storageVersion || 1;
  
  if (currentVersion < STORAGE_VERSION) {
    migrateStorage(currentVersion, STORAGE_VERSION);
  }
});

function migrateStorage(fromVersion, toVersion) {
  console.log(`Migrating storage from v${fromVersion} to v${toVersion}`);
  
  if (fromVersion === 1 && toVersion === 2) {
    // Add new fields to existing calls
    chrome.storage.local.get(null, (items) => {
      const updates = {};
      
      Object.entries(items)
        .filter(([key]) => key.startsWith('call_'))
        .forEach(([key, call]) => {
          call.tags = call.tags || [];
          call.notes = call.notes || '';
          updates[key] = call;
        });
      
      chrome.storage.local.set(updates, () => {
        chrome.storage.local.set({ storageVersion: 2 });
      });
    });
  }
}
```

## Best Practices

```javascript
chrome.storage.local.set(data, () => {
  if (chrome.runtime.lastError) {
    console.error('Storage error:', chrome.runtime.lastError);
```

```javascript
function debouncedSave() {
```

## Troubleshooting

### Quota Exceeded Error
```javascript
chrome.storage.local.set(data, () => {
  if (chrome.runtime.lastError?.message?.includes('QUOTA_EXCEEDED')) {
    console.error('Storage full!');
    
    // Offer cleanup
    if (confirm('Storage is full. Delete old calls?')) {
      cleanupOldCalls(30);
    }
  }
});
```

### Data Corruption
```javascript
// Validate data before saving
function validateCallData(call) {
  return (
    call.callId &&
    call.timestamp &&
    typeof call.duration === 'number' &&
    typeof call.transcript === 'string'
  );
}

if (validateCallData(callData)) {
  chrome.storage.local.set({ [`call_${callData.callId}`]: callData });
} else {
  console.error('Invalid call data:', callData);
}
```

### Lost Data Recovery

```javascript
// Periodic backup to local file

```
